ضمیمہ الف: ہارڈویئر ضروریات اور لیب سیٹ اپ

یہ ضمیمہ 'فزیکل اے آئی اور ہومینوئیڈ روبوٹکس' کورس کے لیے ہارڈویئر کی ضروریات اور مختلف لیبارٹری سیٹ اپ کے اختیارات کی وضاحت کرتا ہے۔ یہ کورس تکنیکی طور پر انتہائی مطالبہ کرنے والا ہے اور فزکس سمیولیشن (Isaac Sim/Gazebo)، بصری ادراک (SLAM/Computer Vision)، اور جنریٹیو اے آئی (LLMs/VLA) کے سنگم پر واقع ہے۔

ہارڈویئر ضروریات کا عمومی جائزہ​

چونکہ کیپ اسٹون پروجیکٹ میں "سیمولیٹڈ ہومینوئیڈ" شامل ہے، اس لیے بنیادی سرمایہ کاری ہائی پرفارمنس ورک اسٹیشنز میں ہونی چاہیے۔ تاہم، "فزیکل اے آئی" کے تصور کو پورا کرنے کے لیے، آپ کو ایج کمپیوٹنگ کٹس (دماغ بغیر جسم) یا مخصوص روبوٹ ہارڈویئر کی بھی ضرورت ہوتی ہے۔

1. "ڈیجیٹل ٹوئن" ورک اسٹیشن (ہر طالب علم کے لیے لازمی)​

یہ سب سے اہم جزو ہے۔ NVIDIA Isaac Sim ایک Omniverse ایپ ہے، جسے RTX (Ray Tracing) سپورٹ کی ضرورت ہوتی ہے۔ عام لیپ ٹاپس (MacBooks یا غیر-RTX Windows مشینیں) کام نہیں کریں گی۔

  • GPU (بڑی رکاوٹ): NVIDIA RTX 4070 Ti (12GB VRAM) یا بہتر۔
    • کیوں: روبوٹ اور ماحول کے USD اثاثے لوڈ کرنے کے لیے زیادہ VRAM درکار ہوتا ہے، اور ساتھ ہی VLA ماڈلز بھی چلانے ہوتے ہیں۔
    • مثالی: RTX 3090 یا 4090 (24GB VRAM) جو "Sim-to-Real" ٹریننگ کو کہیں زیادہ بہتر بناتے ہیں۔
  • CPU: Intel Core i7 (13th Gen+) یا AMD Ryzen 9۔
    • کیوں: Gazebo/Isaac میں فزکس کیلکولیشنز (Rigid Body Dynamics) CPU پر بوجھ ڈالتے ہیں۔
  • RAM: 64GB DDR5 (32GB کم سے کم ہے، مگر پیچیدہ سین رینڈرنگ کے دوران کریش ہوگا)۔
  • OS: Ubuntu 22.04 LTS۔
    • نوٹ: Isaac Sim Windows پر تو چلتا ہے، مگر ROS 2 (Humble/Iron) لینکس پر بہترین چلتا ہے۔ Dual-Boot یا Dedicated Linux مشین لازمی ہے۔

2. "فزیکل اے آئی" ایج کٹ​

چونکہ پورا ہومینوئیڈ روبوٹ مہنگا ہے، اس لیے طلباء پہلے میز پر ہی روبوٹ کا نروس سسٹم تیار کرتے ہیں۔ یہ کٹ Module 3 (Isaac ROS) اور Module 4 (VLA) کے لیے ہے۔

  • دماغ (Brain): NVIDIA Jetson Orin Nano (8GB) یا Orin NX (16GB)
    • کردار: انڈسٹری کا معیار۔ طلباء اپنے ROS 2 نوڈز یہاں ڈپلائے کرتے ہیں۔
  • آنکھیں (Vision): Intel RealSense D435i یا D455
    • کردار: RGB + Depth ڈیٹا فراہم کرتا ہے۔ VSLAM اور Perception کے لیے ضروری۔
  • اندرونی کان (IMU/Balance): USB IMU (BNO055)
  • وائس انٹرفیس: ReSpeaker USB Mic/Speaker Array (Whisper کے لیے)

3. روبوٹ لیب کے اختیارات​

Option A: "پراکسی" روبوٹ (کم بجٹ کے لیے بہترین)​

چوپایہ (dog) یا روبوٹک آرم کو پراکسی کے طور پر استعمال کریں۔

  • روبوٹ: Unitree Go2 Edu ($1,800 – $3,000)
  • فوائد: مضبوط، ROS 2 سپورٹ بہترین، مناسب قیمت۔
  • نقصان: ہومینوئیڈ نہیں ہے۔

Option B: "منی ہومینوئیڈ" طریقہ​

چھوٹے، ٹیبل ٹاپ ہومینوئیڈ روبوٹس۔

  • روبوٹ: Unitree G1 ($16k) یا Robotis OP3 ($12k)
  • کم بجٹ متبادل: Hiwonder TonyPi Pro (~$600)
    • نوٹس: کم قیمت والے روبوٹس Raspberry Pi پر چلتے ہیں، جو Isaac ROS نہیں چلا سکتے۔

Option C: "پریمیم" لیب (Sim-to-Real کے لیے)​

  • روبوٹ: Unitree G1 Humanoid
  • کیوں: بہت کم ایسے کمرشل ہومینوئیڈس ہیں جو حقیقی طور پر چل سکتے ہیں اور ROS 2 انٹیگریشن کھلا رکھتے ہیں۔

4. آرکیٹیکچر کا خلاصہ​

جزوہارڈویئرکردار
Sim RigRTX 4080 + Ubuntu 22.04 PCIsaac Sim، Gazebo، Unity، LLM/VLA ٹریننگ چلاتا ہے۔
Edge BrainJetson Orin Nanoانفیرنس اسٹیک چلاتا ہے، طلباء کا کوڈ یہی رن ہوتا ہے۔
SensorsRealSense + Lidarحقیقی دنیا کا ڈیٹا AI کو فراہم کرتا ہے۔
ActuatorUnitree Go2 یا G1Jetson سے موٹر کمانڈ وصول کرتا ہے۔

"ایتھر" لیب (Cloud-Native) — مہنگا مگر ممکن​

اگر RTX ورک اسٹیشن نہیں ہیں، تو پورا کورس کلاؤڈ پر منتقل کرنا پڑے گا۔

1. کلاؤڈ ورک اسٹیشنز (AWS/Azure)​

  • Instance: g5.2xlarge یا g6e.xlarge
  • سافٹ ویئر: Isaac Sim on Omniverse Cloud
  • لاگت:
    • ~$1.50/گھنٹہ
    • 120 گھنٹے × 1.50 ≈ $180
    • اسٹوریج $25
    • کل ≈ $205 فی سمیسٹر

2. مقامی "بریج" ہارڈویئر​

  • Jetson Kit: اب بھی ضروری ($700)
  • Robot: کم از کم ایک Unitree Go2 ($3,000)

لیٹنسی کا چھپا ہوا مسئلہ​

کلاؤڈ سمیولیشن ٹھیک ہے، لیکن کلاؤڈ سے حقیقی روبوٹ کو کنٹرول کرنا خطرناک ہے۔

  • حل: کلاؤڈ میں ٹریننگ → ماڈل ڈاؤنلوڈ کریں → Jetson پر فلش کریں۔

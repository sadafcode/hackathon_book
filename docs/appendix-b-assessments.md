# Appendix B: Assessments

This appendix outlines the assessment structure and criteria for the 'Physical AI & Humanoid Robotics' course, including the Capstone Project.

## Assessments
- ROS 2 package development project
- Gazebo simulation implementation
- Isaac-based perception pipeline
- Capstone: Simulated humanoid robot with conversational AI

## Capstone Project: The Autonomous Humanoid
A final project where a simulated robot receives a voice command, plans a path, navigates obstacles, identifies an object using computer vision, and manipulates it.
